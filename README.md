# Superstore Sales KPI Dashboard

## Project Overview

This project analyzes sales performance using the Superstore dataset and presents key business insights through an interactive KPI dashboard.

## Objectives

* Analyze revenue and profit trends
* Evaluate category and regional performance
* Identify high-performing products
* Measure business growth using KPIs
* Generate actionable recommendations

## Tools Used

* Power BI
* Excel / CSV
* Data Visualization
* KPI Analysis

## Key Performance Indicators

* Total Revenue: $2.30M
* Net Profit: $286K
* Orders: 5,009
* Year-over-Year Growth: 20.4%

## Dashboard Features

* Revenue Trend Analysis
* Profit Analysis
* Category Performance
* Regional Sales Breakdown
* Customer Segment Analysis
* Strategic Business Insights

## Key Findings

1. Technology category generated the highest profit margin.
2. Furniture category showed the lowest profitability.
3. West region contributed the highest revenue.
4. Consumer segment generated the largest sales volume.
5. Excessive discounting negatively impacted profit margins.
